発表会用のタイマーです。
下段のタイマ設定で時間を入力してください。
発表終了時を「一時停止」にすると、発表時間が来たらカウントが一旦停止しますので再開で質疑をはじめてください。
発表終了時を「継続」にすると一時停止をせずにカウントダウンが続きます。
速度のスイッチは、デバッグ用です。あまり使う事はないでしょう
アラーム音設定は、音を変えたいときに調整できるバーです。
スタートしたら、設定部は非表示になります。
実際使う時は、画面いっぱいに拡大してください。

This is a timer for presentations. Please enter the time in the timer settings at the bottom. If you set the end of presentation to "Pause," the countdown will stop when your presentation time is up, and you can resume it to start the Q&A session. If you set the end of presentation to "Continue," the countdown will continue without pausing. The speed switch is for debugging purposes and you probably won't use it often. The alarm sound setting is a slider that you can use to adjust the sound if you want to change it. The settings section will be hidden once you start the timer. When actually using it, please enlarge it to fill the screen.
